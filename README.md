## 📚 About

Perfect C is a comprehensive C programming learning resource. From basics to advanced concepts, with clear explanations, code examples, and exercises.

## 📋 Topics

| Topic | Description |
|:------|:------------|
| Basics | Variables, data types, operators, I/O |
| Control Flow | if/else, switch, loops |
| Functions | Prototypes, recursion, varargs |
| Arrays & Strings | 1D/2D arrays, string operations |
| Pointers | Pointer arithmetic, function pointers |
| Structures | struct, union, enum, typedef |
| Dynamic Memory | malloc, calloc, realloc, free |
| File Handling | fopen, fread, fwrite, file modes |
| Preprocessor | Macros, #include, conditional compilation |
